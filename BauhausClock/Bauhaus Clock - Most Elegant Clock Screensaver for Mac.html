Frequently asked questions

What makes Bauhaus Clock different from other screensavers?

Bauhaus Clock is the most elegant clock screensaver for Mac which combines precision engineering with stunning visual design. Every element, from clock numerals to movement types, is meticulously crafted to deliver the best clock screensaver experience on a Mac.

It replicates authentic watch mechanics with three movement options: Quartz (precise 1Hz tick), Mechanical (smooth 4Hz sweep at 28,800 beats per hour), and Digital (continuous motion). Features 13 customizable dials with day/night themes, works flawlessly on all Macs (Apple Silicon and Intel) running macOS 12 (Monterey) or later, and includes lifetime updates with your single purchase.

If you value quality and design, this is the clock screensaver for Mac users who appreciate true craftsmanship.

How do I install the Bauhaus Clock screensaver on Mac?

Installing Bauhaus Clock on your Mac is easy:

  1. Download the .saver file and double-click the file to open it.

  2. Click 'Install' when macOS asks you.

  3. Go to System Settings Screen Saver (On macOS 26 Tahoe: Settings Wallpaper Screen Saver).

  4. Select 'Bauhaus Clock' from the list.

  5. Click 'Options' to customize your Bauhaus Clock.

The screensaver is Apple-notarized, so installation is safe and automatic.

Which macOS versions and Macs are supported?

Bauhaus Clock is optimized for macOS 12 (Monterey) and later, and runs on both Apple Silicon and Intel Macs. It's designed to work flawlessly on all Mac displays and external monitors, with perfect scaling and crisp visuals at any resolution.

Does Bauhaus Clock cause screen burn-in on OLED displays?

No, Bauhaus Clock is completely safe for OLED screens. It includes intelligent burn-in protection designed specifically for displays like the MacBook Pro Liquid Retina XDR screen.

The screensaver uses smart algorithms that subtly shift the clock position to prevent any static elements from staying in one place too long. This makes it safe for extended use on MacBook Pro XDR displays and external OLED monitors.

While we've built in comprehensive protection, you're responsible for managing your display settings appropriately.

How do I uninstall or update Bauhaus Clock?

To uninstall the Bauhaus Clock screensaver:

  1. Go to System Settings Screen Saver (On macOS 26 Tahoe: Settings Wallpaper Screen Saver).

  2. Right-click on 'Bauhaus Clock'.

  3. Select 'Delete'.

  4. Confirm removal.

Updating to the latest Bauhaus Clock:

  1. Log into gumroad.com/library

  2. Download the latest Bauhaus Clock files.

  3. Install.

Having issues updating to the latest Bauhaus Clock?

In rare occasions coming from an earlier version of Bauhaus Clock, the install of the latest version can run into issues.

Following these steps in exact order solves 99% of update problems:

  1. Go to System Settings Screen Saver Right-click "Bauhaus Clock" Delete (On recent macOS versions: Settings Wallpaper Screen Saver).

  2. Restart your Mac (this is important).

  3. Install the latest version.
